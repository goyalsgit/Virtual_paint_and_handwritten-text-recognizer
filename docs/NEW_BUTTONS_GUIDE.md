## 🎮 How Each Button Works

### 1. 🔍+ Zoom In Button

**What it does:**
- Makes PDF bigger by 20%
- Maximum zoom: 300%

**How to use:**
```
1. Point index finger at "🔍+ Zoom In" button
2. Hold for 0.5 seconds
3. PDF zooms in
4. Pill shows: "🔍+ Zoomed to 170%"
```

**Example:**
```
Current: 150% → Click Zoom In → New: 170%
Current: 170% → Click Zoom In → New: 190%
Current: 280% → Click Zoom In → New: 300% (max)
```

## 🔢 Technical Details (Simple Explanation)

### How Zoom Works
```javascript
// Current zoom level stored in variable
let scale = 1.5; // 150%

// Zoom In: Add 0.2 (20%)
scale = scale + 0.2; // Now 170%

// Zoom Out: Subtract 0.2 (20%)
scale = scale - 0.2; // Now 130%

// Limits
MIN_SCALE = 0.5  // 50% minimum
MAX_SCALE = 3.0  // 300% maximum
```

### How Brush Size Works
```javascript
// Current brush size stored in variable
let currentBrushSize = 3; // 3 pixels

// Bigger: Add 2 pixels
currentBrushSize = currentBrushSize + 2; // Now 5px

// Smaller: Subtract 2 pixels
currentBrushSize = currentBrushSize - 2; // Now 1px

// Limits
Minimum = 1 pixel
Maximum = 20 pixels
```

### How Highlighter Works
```javascript
// Highlighter is just a tool mode
activeTool = "highlighter";

// When drawing:
if (activeTool === "highlighter") {
  // Use thick, transparent lines
  lineWidth = 18;
  opacity = 0.35;
} else if (activeTool === "pen") {
  // Use thin, solid lines
  lineWidth = currentBrushSize;
  opacity = 0.85;
}
```

## 📖 Code Explanation (For Beginners)

### Where Buttons Are Defined (HTML)
```html
<!-- This creates the button -->
<div class="v-btn v-zoom-btn" data-action="zoom-in">
  🔍+ Zoom In
</div>

Explanation:
- class="v-btn" → Makes it a virtual button
- class="v-zoom-btn" → Gives it blue color
- data-action="zoom-in" → Tells JavaScript what to do
- 🔍+ Zoom In → Text shown on button
```

### How Button Click Works (JavaScript)
```javascript
// Step 1: User hovers button
// Step 2: JavaScript detects hover
// Step 3: Check what button was hovered

if (action === "zoom-in") {
  // Step 4: Increase zoom
  scale = scale + 0.2;
  
  // Step 5: Redraw PDF at new size
  rerenderAtScale();
  
  // Step 6: Show message
  showPill("🔍+ Zoomed to 170%");
}
```

### How Brush Size is Stored
```javascript
// Global variable (accessible everywhere)
let currentBrushSize = 3;

// When user clicks "Bigger"
currentBrushSize = currentBrushSize + 2;

// When drawing, use this size
ctx.lineWidth = currentBrushSize;
```

## 🐛 Troubleshooting

### Issue: Zoom doesn't work
**Check:**
- Is PDF loaded?
- Are you hovering long enough?
- Check browser console for errors

**Solution:**
```javascript
// Verify scale variable exists
console.log("Current scale:", scale);

// Verify function exists
console.log("rerenderAtScale:", typeof rerenderAtScale);
```

### Issue: Brush size doesn't change
**Check:**
- Is pen or highlighter mode active?
- Is currentBrushSize variable defined?

**Solution:**
```javascript
// Check current size
console.log("Brush size:", currentBrushSize);

// Manually set size
currentBrushSize = 5;
```

### Issue: Highlighter not working
**Check:**
- Is button visible?
- Is setActiveTool function working?

**Solution:**
```javascript
// Manually activate highlighter
setActiveTool("highlighter");
```
